# Training Needs Gap Analysis
## Version: v1.0

**Workflow:** L&D / Workforce Development — Quarterly workflow. Triggered when annual competency assessment results are uploaded to the LMS.

**Problem Being Solved:** L&D managers spend 6–8 hours per quarter manually cross-referencing individual competency results against role requirements. For a 500-staff hospital, this is a significant planning bottleneck.

**Automation Potential:** Medium — AI analyses structured assessment data and generates a prioritised gap analysis report by department. Estimated 70% reduction in analysis time.

**Key Risks:** Errors in source data produce misleading analysis; individual staff must not be identified; AI may present uncertain patterns with false confidence.
